# Week8 グループワーク：かんたん温度モニタ作成

## 🎯 目的（40分）
牛舎・鶏舎などの温度データ（数値リスト）を使って、サンプルコードの空欄を埋めながら以下を実装します：

1. 平均気温を計算する  
2. 最も気温が高い時間（時刻）を求める  
3. 気温がしきい値以上になった「アラート時間数」を数える  
4.（発展）1日のレポート行を自動生成する  

外部ファイルの読み込みやグラフ描画は行いません。  
すべて **Python の基本文法（リスト・for 文・if 文・関数）だけで完結**します。

---

## 📌 この演習の意義
平均値・最大値・しきい値判定などの処理は、  
**センサー処理・アンケート集計・品質管理など、ほぼ全ての実データ処理の基礎**となります。
